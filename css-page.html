Strona główna HTML CSS Notatki

CSS

Typy własności display

display: block
- Elementy ustawiają się jeden pod drugim
- Zajmują całą szerokość okna

display: inline
- Kurczy się do zawartości
display: inline-block
- umożliwia dodawanie dodatkowych własności, np. width i height

DISPLAY

display: block
- min-width, width, max-width
- min-height, height, max-height
- overflow: hidden (domyślnie visible); - ucinanie wystającej zawartości
- overflow: auto; - dodaje scrolla w elemencie


Jednostki

- 0
- piskele (px)
- procenty (%) - liczone względem rodzica
- vw (viewport width) - szerokość elementu względem okna
- vh (viewport height) - wysokość elementu względem okna

BOX MODEL

MARGIN->BORDER->PADDING->ELEMENT
padding
padding - wewnątrzny margines,wg. wskazówek zegara (top-right-bottom-left) lub lustrzanie (top+bottom/right/left)
border
- border-width: 10px
- border-style: solid, dashed, dotted
- border-color: blue
- lub - border: 10px solid blue
- można też - border-top(left,right,bottom): 10px solid blue
- border-radius: 20px (zaokrąglenie ramki)
margin
margin: 10px auto (własność auto wyśrodkowuje element na stronie - działa tylko przy zdefiniowanej szerokości elementu)

BOX-SIZING

border-box
box-sizing: content-box (default)
box-sizing: border-box

POSITION

position: static (default)
position: relative - otwiera możliwość korzystania z absolute
position: absolute - otwiera możliwośc pozycjonowania względem elmentu z własnością "relative" (np. top, right, bottom, left: 20px)
position: fixed - pozwala ustalać niezmienną pozycję względem okna position: sticky - przykleja się i przesuwa, ale nie może opuścić rodzica (nie może mieć display:overflow) z-index: (liczba całkowita) - ustala kolejność wyświetlania

FLOAT

float: right; - wszystko omywa element z lewej strony float: left; - wszystko omywa element z prawej strony clear: left (right, both) - czyści float, "rozszerza pudełko"

LIST

- list-style - ul - unordered list - ol - lista zagnieżdżona - dl - description list - lista opisowa - dt - description term - opisywany termin - dd - description details - opis terminu
